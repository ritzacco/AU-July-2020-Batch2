FlatMap

Conversion of an observable of observables into a single observable i.e. 'flattening' is called FlatMap.. For RxJs, the mapping function looks like this: item => Observable. rather than this: item => Array.

SwitchMap

A switchMap operator will return an immediately reflected observable which subequently emits items on the basis of a function specified to each item that is emitted by the "source" observable where that function returns the inner observable.

ExhaustMap

ExhaustMap returns an observable that emits items based on a function that is specified to each item emitted by the source Observable, where that function returns an (so-called "inner") Observable. On projecting a source value to an Observable, the output Observable begins emitting the items emitted by that projected Observable.

ConcatMap

The Angular ConcatMap maps each value from the source observable into an inner observable, subscribes to it, and then starts emitting the values from it replacing the original value. It creates a new inner observable for every value it receives from the Source.

ViewEncapsulation

Defines template and style encapsulation options available for Component's Component.

Members

  • Emulated : Emulate Native scoping of styles by adding an attribute containing surrogate id to the Host Element and pre-processing the style rules provided via styles or styleUrls, and adding the new Host Element attribute to all selectors. This is the default option.
  • None: Don't provide any template or style encapsulation.
  • ShadowDom: Use Shadow DOM to encapsulate styles.
